# Aspirational vs implemented

Unless a proposal or ADR **explicitly** maps to a concrete path under `src/`, `tests/`, or a running env var **with tests**, treat language about **consciousness, soul, sentience, phenomenology, or aliveness** as **vision / metaphor / research framing**, not a claim about the current software artifact.

**Implemented** means: code + tests (or a documented, reproducible script) in this repository. Everything else is roadmap or design exploration.
