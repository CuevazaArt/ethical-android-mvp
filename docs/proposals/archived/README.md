# Archived proposals

This directory holds **superseded** or **historical** design notes that were moved from `docs/proposals/` to reduce root clutter. Moves are done **incrementally** so inbound links from issues and older commits can be updated in small PRs.

Do **not** delete the originals from git history; use `git mv` when moving a file, and add a one-line pointer stub at the old path if external URLs must keep resolving.

See [../INDEX.md](../INDEX.md) and [../PLAN_WORK_DISTRIBUTION_TREE.md](../PLAN_WORK_DISTRIBUTION_TREE.md) (Bloque 36.0).
