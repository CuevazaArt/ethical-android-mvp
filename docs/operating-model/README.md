## Export to another project

1. Copy this folder and all referenced collaboration files.
2. Adapt branch names, role names, and governance authority labels.
3. Keep invariants: evidence-based closure, protected production, and quality gates.
4. Add your CI decision matrix (full checks vs docs-only skip) before rollout.

For an implementation checklist, see [Import and Adoption Guide](IMPORT_AND_ADOPTION.md).
