# Companion binaries (not tracked in git)

The repository [`.gitignore`](../../../.gitignore) excludes `*.pdf` and `*.docx`. The former **`prealphaDocs/`** drop folder sometimes included these alongside the markdown and media now under [`docs/historical/prealpha/`](README.md). If you maintain a local backup, you may still have:

- `mente_androide_civico_paper (1).pdf`
- `modelo_conciencia_ascii_1.1.pdf`
- `modelo_conciencia_bayes.pdf`
- `paper_modelo_conciencia_ascii.pdf`
- `serArtificial1.3.pdf` … `serArtificial1.6` (various revisions)

**Word**

- `androide_etico_alpha.docx` (export of the alpha document)
- `bibliografia_androide_etico.docx`
- Empty or placeholder `.docx` files have **no** archival value.

**Policy:** Do not commit large binary duplicates here unless the project changes `.gitignore` policy deliberately. Prefer linking to published PDFs or keeping exports in foundation archives outside the repo.
