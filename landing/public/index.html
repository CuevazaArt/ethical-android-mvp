What it does

A lean Python kernel that reasons over ethical precedents, perceives intent deterministically (<1ms, no LLM), and runs on any Android device — local-first, no cloud required.

⚖️

Case-Based Ethics (CBR)

36 richly annotated precedent cases. Each decision is matched against real ethical scenarios — not a black-box score. Every answer is traceable to a case.

🧠

Deterministic perception

Intent, risk, and hostility classified in <1ms with zero LLM dependency in the hot path. The ethical gate is never slowed by model latency.

🔒

MalAbs safety gate

Lexical absolute-evil filter before anything enters the kernel. Adversarial suite: 6/6 harmful blocked · 10/10 legitimate pass. Cannot be bypassed by prompt injection.

🌐

WebSocket chat server

FastAPI + uvicorn; streaming JSON over /ws/chat. Per-session kernel state, async timeout, and observable JSON output on every turn.

📱

Native Android SDK

Kotlin / Jetpack Compose foreground service. Exposes device capabilities (camera, mic, NPU) to the Ethos mesh. BSL 1.1 → Apache 2.0 after 36 months.

🔌

Extensible plugin system

Built-in: Time, System, Weather (wttr.in), Web search (DuckDuckGo). Plugins run in the chat pipeline with full STM continuity — no context break.

🔐

Secure Vault

Encrypted secret store with WebSocket authorization pipeline. Sensitive state is isolated behind a runtime permission barrier.

🪞

Narrative identity

Evolving LLM journal with archetype distillation and Psi-Sleep REM consolidation. The kernel reflects on its own history every 120 s of idle.

📊

Real-time dashboard

Open dashboard.html for live ethical-pole radar charts, perception confidence, governance state, and chat.

Quick start

Python 3.11+ recommended. CI runs on 3.11 and 3.12. Ollama optional (heuristic fallback available).

# 1. Clone and install git clone https://github.com/CuevazaArt/ethical-android-mvp.git cd ethical-android-mvp python -m venv .venv && source .venv/bin/activate pip install -r requirements.txt pip install -r requirements-dev.txt # for tests + lints
# 2. Run the chat server python -m src.chat_server # → http://127.0.0.1:8000/ WebSocket: ws://127.0.0.1:8000/ws/chat # → http://127.0.0.1:8000/dashboard Real-time ethical dashboard
# 3. Open the dashboard (no npm needed) # Serve landing/public/ with any static server, e.g.: python -m http.server 9000 --directory landing/public # → open http://localhost:9000/dashboard.html
# 4. Run the test suite pytest tests/core/ -q # → 203 tests · 0 failures python -m ruff check src tests python -m mypy src
# 5. GPU deploy (NVIDIA + Ollama sidecar) docker compose -f docker-compose.gpu.yml up --build # → Ollama model pulled automatically; kernel uses GPU inference

Architecture overview

V2 replaced the distributed tri-lobe event-bus with a direct sequential pipeline inside ChatEngine. Every decision is deterministic and traceable. LLM is an optional accelerator — never the ethical authority.

Input text │ ▼ MalAbs gate ←── Lexical absolute-evil filter (deterministic, <1ms) │ PASS ▼ Deterministic perception ←── Intent · risk · hostility [0,1], no LLM │ ▼ CBR Ethics ←── 36 annotated precedent cases, cosine similarity match │ Returns: action · confidence · matched_case ▼ LLM respond ←── Optional: generates verbal reply from kernel decision │ <think> tags silenced by state machine ▼ Memory / Identity / Roster ←── Hybrid embeddings + TF-IDF │ Narrative journal · PersonCard context ▼ WebSocket JSON stream ←── plugin_used · ethics_case · latency · perception

Psi-Sleep daemon: After 120 s idle the kernel enters REM mode — consolidates memory, distills archetype narrative, prunes low-weight episodes.

Android SDK (V2.77+): Kotlin / Jetpack Compose foreground service (src/clients/nomad_android/). Exposes CameraX, AudioRecord (PCM 16kHz), and NPU inference to the Ethos mesh via P2P discovery (V2.79). See Android Studio handoff guide.
