WebSocket chat (smoke test)

Start the server from the repo root: python -m src.chat_server
Override bind with env CHAT_HOST / CHAT_PORT (default 127.0.0.1:8765).

Opening this file as file://: some browsers block WebSockets; if it fails, serve this folder (e.g. cd landing/public && python -m http.server 8080) and open http://127.0.0.1:8080/chat-test.html. The FastAPI app does not need CORS for same-machine ws:// from a local page.
