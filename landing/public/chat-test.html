WebSocket chat (smoke test)

PC: python -m src.runtime with CHAT_HOST=0.0.0.0 for LAN — see repo docs/LOCAL_PC_AND_MOBILE_LAN.md or run scripts/start_lan_server.ps1 / start_lan_server.sh. Phone UI: mobile.html

Phone / same WiFi: serve this folder (python -m http.server 8080 --bind 0.0.0.0 in landing/public), then open http://<PC_IP>:8080/chat-test.html?host=<PC_IP> — WebSocket URL is filled below.

file://: some browsers block WebSockets; use HTTP as above. Query: ?url=ws://... full override, or ?host=192.168.x.x&port=8765.
