# Local LLM: Ollama

The Ethos Kernel documents **[Ollama](https://ollama.com/)** as the primary open-source path for natural-language layers (`LLM_MODE=ollama`, `USE_LOCAL_LLM`, `OLLAMA_BASE_URL`, `OLLAMA_MODEL`). See the repository root [`README.md`](../README.md#real-time-chat-websocket).
